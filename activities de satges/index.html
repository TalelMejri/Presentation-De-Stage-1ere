Lors de la visite de ce service j'étais informé tout d’abord sur le système national d’encaissement SNE :
Ce système est une Platform qui permet de réunir le travail à travers une connexion commune à un réseau
pour une exploitation d’une unique base de données.



Cette opération comporte plusieurs étapes :


  1. Réception des factures.
  2. Classer par date.
  3. Saisie manuelle (Par saisir numéro de téléphone, le montant, produit, code encaissement).
  4. Traitement des anomalies.
  5. Clôture du journal comptable.
  6. Clôture du mois comptable.
  7. Edition du certificat annuel d’encaissement
Un Système de télécommunications est un ensemble de liaisons et de fonctions permettant d'assurer un service.
Un réseau (Network) : est un ensemble d'objets interconnectés les uns avec les autres.
Il permet de faire circuler des éléments entre chacun de ces objets selon des règles bien définies.
L’utilité de se connecter à un réseau est le partage facile des données (fichier, applications…),
leur manipulation ainsi que la communication entre processus.
On distingue différents types de réseaux :
LAN (Local Area Network) :
réseau local est un réseau informatique à une échelle géographique relativement restreinte, il est utilisé pour relier entre eux les ordinateurs : par exemple d'une entreprise, d'une salle informatique
WAN (Wide Area Network) :
il s’agit d’un réseau étendu c’est à dire un réseau informatique (ou de télécommunications) couvrant une grande zone géographique. (Pays...).
MAN : métropolitain area network
permet d’interconnecter des utilisateurs et des ressources informatiques dans une zone ou une région géographique plus grande que celle couverte par un réseau local (LAN).
L’objectif de cette application est d’enregistrer les données de tous les employés de TT dans une base de donnes plus sécurisé, elle doit fournir les services suivants :
  1. Eviter la redondance des éléments stockés.
  2. Possibilité de gérer le matériel à plusieurs, via toute machine connectée au réseau.
  3. Permet de répertorier le tout dans la base de données.
